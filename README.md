# Machine_Learning_for_Smarter_Inventory_Management

## ğŸ“Œ Project Introduction

This project is an e-commerce web application designed to provide a modern online shopping experience. The platform supports **three main roles**:

- **Buyer**: Can browse products, view details, and receive personalized product recommendations.  
- **Seller**: Can add and manage their products across seven categories: Books, Food & Drinks, Accessories, Clothing, Electronics, Household Items, and Skincare.  
- **Admin**: Manages users, products, and overall platform operations.  

In addition to standard e-commerce features, the website incorporates **machine learning** to enhance user experience and business operations:

- **Sales Forecasting**: Predicts future demand for products to help sellers and admins manage inventory efficiently.  
- **Product Recommendation System**: Suggests relevant products to buyers based on their browsing and purchase history, increasing engagement and sales.

The project emphasizes **frontend development with React**, a **clean and responsive UI**, and a **scalable architecture** to support future data-driven enhancements and multi-role functionalities.

## How to run the project

Clone the Repository
``` 
git clone https://github.com/MaoMonioudom/Machin_Learning_for_Smarter_Inventory_Management
```
Go into the project
```
cd Machin_Learning_for_Smarter_Inventory_Management
```
Install dependency 
```
npm install react react-router react-dom react-router-dom react-icons
```
Start the server
```
npm run dev
```
Open Browser the visit
```
http://localhost:5173
```
